# Multi-Threaded Vulnerability Scanner
A professional-grade network reconnaissance tool developed to automate service identification and vulnerability mapping.

### Key Features
* **Multi-threaded Port Scanning:** Utilizes Python `threading` and `queue` for high-speed asset discovery.
* **Service Fingerprinting:** Interrogates open ports to capture service banners.
* **Vulnerability Mapping:** Cross-references service versions against a local CVE database.
* **Automated Reporting:** Generates timestamped scan reports in `.txt` format.

### Tech Stack
Python (Socket, Threading, JSON, Regex)
